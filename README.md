# ✅ Todo List App

A modern and fully functional task management app with filtering, completion tracking, and a vibrant blue gradient background.

---

## 📸 Preview

> A white card with a task input field, filter tabs (All / Pending / Completed), and a list of tasks with checkboxes and a three-dot menu per item.

---

## ✨ Features

- ➕ Add new tasks via the input field
- ☑️ Mark tasks as **complete** (renders with strikethrough)
- 🔍 Filter tasks by:
  - **All** — shows every task
  - **Pending** — shows incomplete tasks only
  - **Completed** — shows finished tasks only
- 🗑️ **Clear All** button to remove all tasks at once
- ⋮ Three-dot menu per task for edit/delete options

## 🚀 Getting Started

1. Open `index.html` in any modern browser
2. Type a task in the input field and press **Enter**
3. Check the checkbox to mark a task complete
4. Use the filter tabs to view tasks by status
5. Click **Clear All** to reset the list

> No dependencies or build tools required!
